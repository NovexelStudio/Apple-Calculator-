## About

An Apple-style calculator application for Android that recreates the clean iOS calculator experience with a focus on:

- Pixel-perfect circular buttons  
- Orange operator accents  
- Smooth ripple animations  
- Dark theme interface  
- Clean and modular Kotlin logic  

Built entirely using:

- Kotlin  
- XML Layouts  
- ConstraintLayout  
- Android Studio  

---

## Features

- Splash (Prestige) screen  
- Basic arithmetic operations  
- Responsive layout design  
- Smooth touch feedback  
- Minimal UI hierarchy
